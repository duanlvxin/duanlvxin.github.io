**二分和回溯在代码书写上是有明确的公式的。**

## 二分

前提：有序数组
思想：判断中间位置的值大小，对半分去找

```js
function search(arr, target) {
  let low = 0, high = arr.length - 1
  while(low <= high) {
    let mid = low + ((high - low) >> 1)
    if(arr[mid] === target) {
      return mid;
    }else if(arr[mid]<target){
      low = mid + 1;
    } else {
      high = mid -1;
    }
  }
  return -1;
}
```
